# Three Good Things (3つの良いこと記録アプリ)

このアプリケーションは、一日の終わりに「3つの良いこと」を記録するためのシンプルなWebアプリです。自己肯定感を高め、ポジティブな視点を持つための習慣をサポートします。

## 技術スタック

- **フレームワーク**: Next.js (App Router)
- **言語**: TypeScript
- **データベース**: Dexie.js (IndexedDB) - ブラウザ内にデータを保存するため、サーバーなしで動作します。
- **UI コンポーネント**: Lucide React (アイコン)
- **スタイリング**: Tailwind CSS

## ディレクトリ構成

- `page.tsx`: メインのレイアウトとタブ（今日、履歴）の切り替えを管理します。
- `db.ts`: データベース（Dexie.js）の設定とデータ型定義を行います。
- `components/DailyEntryForm.tsx`: 今日の3つの良いことを入力・編集するためのフォームです。
- `components/HistoryList.tsx`: 過去の記録を一覧表示し、削除する機能を提供します。

## 主要な機能

### 1. オフライン対応データ保存
Dexie.js を使用して、ユーザーのブラウザ（IndexedDB）にデータを保存します。これにより、データはデバイス内に留まり、プライバシーが守られるとともに、インターネット接続なしでも動作します。

### 2. 今日の記録 (DailyEntryForm)
- その日の日付（`YYYY-MM-DD`）をキーにしてデータを管理します。
- ページを開いた際、既にその日の記録があれば自動的に読み込みます。
- 保存ボタンを押すと、既存データがあれば更新（Update）、なければ新規作成（Add）を行います。

### 3. 履歴管理 (HistoryList)
- 過去のすべての記録を日付の降順（新しい順）で表示します。
- 必要なくなった記録を個別に削除することができます。
- `useLiveQuery` フックを使用しているため、データが変更されるとリアルタイムで一覧が更新されます。

## データ構造

`DailyEntry` インターフェース:
- `id`: UUID (ユニークID)
- `date`: 日付文字列 (`YYYY-MM-DD`)
- `thing1`, `thing2`, `thing3`: 記録内容
- `createdAt`: 作成日時（タイムスタンプ）
- `updatedAt`: 更新日時（タイムスタンプ）
